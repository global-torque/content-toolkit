# Changelog

  SHA-pinned CI, public API governance, clean-source artifact manifests, and
  provenance workflow.
- Removed a backtracking terminal-slash expression from caller-controlled path
  normalization and added an adversarial long-separator regression.
- Supersedes the dirty-tree beta.6 implementation artifact; beta.6 remains
  historical local evidence and must not be uploaded or retagged.

## 0.2.0-beta.5 - Unreleased

- Inlined the distributive typed-search helper so the committed public API
  report contains no forgotten-export warning.

## 0.2.0-beta.4 - Superseded local candidate

- Required plain-object root frontmatter and guaranteed that derived root
  overrides survive omitted non-enumerable source properties.
- Locked comparator inputs before host sorting, froze implementation
  prototypes, and captured default index-page policy without retaining the
  caller's mutable options object.
- Preserved tuple shapes in deep-readonly types, returned frozen path segments,
  rejected boxed and non-string custom-normalizer results, and used the
  intrinsic Date timestamp operation while sorting hostile input records.
- Made node ownership unforgeable, froze the reachable implementation
  constructors, rejected prototype-created or foreign scope nodes, and clamped
  node-level searches to their receiver subtree.
- Hardened the shared publication gates to reject private dependency specs,
  product-specific packed content, and source maps that escape the package.
- Accepted genuine cross-realm dates and plain records, and preserved the
  package's immutable date views across repeated normalize, tree, and sort
  pipelines.
- Promoted and validated own semantic `url`, `rawUrl`, `draft`, and configured
  path/source descriptors without executing accessors; malformed visibility or
  path values now fail closed.
- Supported ordinary closed interfaces and open parsed-frontmatter records,
  while restricting closed-interface path keys to optional or required string
  fields.
- Made scalar filters use SameValueZero, added immutable publish-date typing,
  and kept indexed and scan lookup behavior aligned for `NaN` and path values.
- Resolved exact canonical tree/index keys before non-idempotent host
  normalization, retained default URL indexes, and made stored node URLs
  self-resolving across repeated terminal `index` paths.
- Returned `null` instead of self-navigation for singleton sibling sets and
  introduced node-only find options that cannot escape the receiver subtree.
- Rejected path-segment boundary whitespace, captured mutable callback policy
  once, and eliminated formatter/srcset time-of-check/time-of-use re-reads.
- Froze every split-path result and preserved hidden custom semantic fields,
  sparse arrays, root cycles, symbols, and safe dangerous own keys.
- Never publish this candidate. Its generated public API report contained an
  `ae-forgotten-export` warning for the private `ContentFieldMember` helper.
  Its local tarball SHA-512 was
  `dd38aee62cf630935f4378b56d8cf0b213b0e9efca797b39b228b838809db307c2fc1adefe4fd85947586c7119cc701d0b88204eb3c15c231c9e8bb6f2fa216e`.

## 0.2.0-beta.3 - Superseded local candidate

- Preserved sparse-array indices, root cycles, enumerable symbols, and
  dangerous own keys without attacker-controlled prototype changes.
- Rejected accessors before draft/path inspection, aligned indexed/scan
  equality and foreign-node scope, and restricted indexes to primitive values.
- Normalized required and scan lookup values once and hardened transformed
  segments/tree paths against browser separator and dot canonicalization.
- Hid mutable tree indexes behind native private fields, froze tree/node
  facades, and rejected calls to construction-only mutation hooks.
- Treated blank path-index values as absent and cloned dates through the Date
  intrinsic without executing hostile own accessors.
- Failed unresolved API Extractor exports explicitly.

  mutable prototypes could bypass runtime immutability, and the default
  index-page normalizer retained caller-owned options. Its local tarball
  SHA-512 was
